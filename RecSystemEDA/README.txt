This folder contains the initial analysis done on the data.
There are three main datasets:
-Ratings: contains 3 column pertaining to the UserID, MovieID, and the rating given to the movie and a timestamp column
-Movies: contains 4 columns pertaining to MovieID, Title, Genre, and Year of release
Rmat: this is a csv file with 6040 MovieID as columns, and 3706 UserID as rows


This analysis contains two sections:

System 1: Recommends the top ten most popular movies based on their average ratings.

System 2: Recommendations made using Item Based Collaborative Filtering (IBCF)
